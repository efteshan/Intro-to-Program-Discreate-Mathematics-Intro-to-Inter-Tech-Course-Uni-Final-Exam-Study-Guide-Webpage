Questions 1 to 10 · Union, Intersection, Complement, Power Set
Q1 Prove: A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) (Distributive Law) ▾
💡 What this is asking
This is the Distributive Law of Set Theory. It says "union distributes over intersection", just like multiplication distributes over addition in algebra (a×(b+c) = a×b + a×c). We prove it by showing every element on the left side is also on the right, and vice versa.
✅ Proof
  1. Forward direction: Let x ∈ A ∪ (B ∩ C). Then x ∈ A, or x ∈ (B ∩ C).
  2. Case 1: x ∈ A → x ∈ A ∪ B AND x ∈ A ∪ C → x ∈ (A ∪ B) ∩ (A ∪ C)
  3. Case 2: x ∈ B ∩ C → x ∈ B AND x ∈ C → x ∈ A ∪ B AND x ∈ A ∪ C → same conclusion
  4. Backward direction: Let x ∈ (A∪B) ∩ (A∪C). Then x ∈ A∪B and x ∈ A∪C.
  5. If x ∈ A → done. If x ∉ A, then x must be in B (from A∪B) AND in C (from A∪C) → x ∈ B∩C → x ∈ A∪(B∩C)
∴ A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C) ✓ (Proven by element argument)
🎯 Practice Questions
📝 Try These
P1. Prove: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
Similar element argument: If x ∈ A∩(B∪C), then x ∈ A and (x ∈ B or x ∈ C).
Case 1: x ∈ B → x ∈ A∩B → x ∈ (A∩B)∪(A∩C)
Case 2: x ∈ C → x ∈ A∩C → x ∈ (A∩B)∪(A∩C)
Backward: if x ∈ (A∩B)∪(A∩C) → x ∈ A∩B or x ∈ A∩C → x ∈ A and x ∈ B∪C ✓
P2. Using A={1,2,3}, B={2,3,4}, C={3,4,5}, verify: A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
B ∩ C = {3,4}
LHS: A ∪ {3,4} = {1,2,3,4}
A ∪ B = {1,2,3,4}, A ∪ C = {1,2,3,4,5}
RHS: {1,2,3,4} ∩ {1,2,3,4,5} = {1,2,3,4} ✓ Both sides equal.
Q2 Verify De Morgan's Law: (A ∩ B)' = A' ∪ B' ▾
💡 What this is asking
De Morgan's Laws are two rules: (A∩B)' = A'∪B' and (A∪B)' = A'∩B'. They tell us how complements work with ∩ and ∪. Think of it like English: "NOT (raining AND cold)" = "NOT raining OR NOT cold".
✅ Verification with example
Let U={1,2,3,4,5}, A={1,2,3}, B={2,3,4}
  1. A ∩ B = {2,3}
  2. (A ∩ B)' = U − {2,3} = {1,4,5} ← Left side
  3. A' = U − A = {4,5} and B' = U − B = {1,5}
  4. A' ∪ B' = {4,5} ∪ {1,5} = {1,4,5} ← Right side
Both sides = {1,4,5} ✓ De Morgan's Law verified!
🎯 Practice Questions
📝 Try These
P1. Verify the other De Morgan's Law: (A ∪ B)' = A' ∩ B' using U={1..6}, A={1,2,3}, B={3,4,5}
A ∪ B = {1,2,3,4,5} → (A∪B)' = {6} ← LHS
A' = {4,5,6}, B' = {1,2,6}
A' ∩ B' = {6} ✓ Both sides equal.
P2. If U={a,b,c,d,e}, A={a,c,e}, B={b,c,d}, find (A∩B)' and A'∪B' and verify they're equal.
A∩B = {c} → (A∩B)' = {a,b,d,e}
A' = {b,d}, B' = {a,e}
A'∪B' = {a,b,d,e} ✓ Verified.
Q3 A={1,2,3,4}, B={3,4,5,6}: Find A∪B, A∩B, A−B ▾
💡 What this is asking
Union (∪) = all elements in EITHER set. Intersection (∩) = elements in BOTH sets. Difference (A−B) = elements in A but NOT in B. These are the 3 most fundamental set operations.
✅ Answer
  1. A ∪ B = all elements from both = {1,2,3,4,5,6}
  2. A ∩ B = only what's in both = {3,4}
  3. A − B = in A but remove what's in B = {1,2,3,4} − {3,4} = {1,2}
A∪B = {1,2,3,4,5,6} | A∩B = {3,4} | A−B = {1,2}
🎯 Practice Questions
📝 Try These

A∪B = {2,4,6,8,10,12}
A∩B = {4,8}
B−A = {10,12}
P2. A={a,b,c,d}, B={c,d,e,f}. Find A∩B and A−B.
A∩B = {c,d}
A−B = {a,b}
Q4 Find the Power Set of A = {2, 4, 6} ▾
💡 What this is asking
The Power Set P(A) is the set of ALL possible subsets of A, including the empty set and A itself. If |A| = n elements, then |P(A)| = 2ⁿ subsets. Here n=3, so we get 2³ = 8 subsets.
✅ Answer
A = {2, 4, 6} → n = 3 → 2³ = 8 subsets
P(A) = { ∅, {2}, {4}, {6}, {2,4}, {2,6}, {4,6}, {2,4,6} }
🎯 Practice Questions
📝 Try These
P1. Find the power set of B = {a, b}. How many subsets?
n=2 → 2² = 4 subsets
P(B) = {∅, {a}, {b}, {a,b}}
P2. If |P(A)| = 16, how many elements does A have?
2ⁿ = 16 → n = 4. So A has 4 elements.
Q5 Find A ∩ (B ∪ C) with A={1,2,3}, B={2,3,4}, C={3,4,5} ▾
💡 What this is asking
This is a nested set operation. Always work inside out: compute B∪C first, then intersect the result with A. Like maths with brackets: do brackets first!
✅ Answer
  1. B ∪ C = {2,3,4} ∪ {3,4,5} = {2,3,4,5}
  2. A ∩ (B∪C) = {1,2,3} ∩ {2,3,4,5} = {2,3}
A ∩ (B ∪ C) = {2, 3}
🎯 Practice Questions
📝 Try These

B∪C={1,3,4,5}. A∩{1,3,4,5}={1,3,4}
P2. A={x,y,z}, B={y,z,w}, C={z,w,v}. Find (A∪B) ∩ C.
A∪B={x,y,z,w}. {x,y,z,w}∩{z,w,v}={z,w}
Q6 Check A ⊆ B: A={2,3}, B={1,2,3,4,5} ▾
💡 What this is asking
A is a subset of B (written A ⊆ B) if every element of A is also in B. Check each element of A one by one.
✅ Answer
  1. Is 2 ∈ B? Yes (B = {1,2,3,4,5})
  2. Is 3 ∈ B? Yes
A ⊆ B ✓ Yes, A is a subset of B.
🎯 Practice Questions
📝 Try These
P1. Is A={1,5,7} ⊆ B={1,2,3,4,5,6}?
7 ∉ B. So A ⊄ B. No, A is NOT a subset of B.
P2. Is ∅ a subset of every set? Explain.
Yes. The empty set is a subset of every set because there are no elements in ∅ to violate the condition. This is called a "vacuous truth."
Q7 Find A × B (Cartesian Product): A={1,2}, B={x,y} ▾
💡 What this is asking
The Cartesian Product A×B is every possible ordered pair (a, b) where a comes from A and b comes from B. If |A|=m and |B|=n, then |A×B| = m×n pairs.
✅ Answer
Pair every element of A with every element of B:
A × B = {(1,x), (1,y), (2,x), (2,y)}
Note: 2 × 2 = 4 ordered pairs total
🎯 Practice Questions
📝 Try These
P1. A={a,b,c}, B={1,2}. Find A×B and |A×B|.
A×B = {(a,1),(a,2),(b,1),(b,2),(c,1),(c,2)}. |A×B| = 6

A×B={(1,p),(1,q),(2,p),(2,q)}
B×A={(p,1),(p,2),(q,1),(q,2)}
They are NOT equal (order matters in pairs). A×B ≠ B×A in general.
Q8 If n(A)=6, n(B)=5, n(A∩B)=2, find n(A∪B) ▾
💡 What this is asking
This uses the Inclusion-Exclusion Principle: when you add two sets together, you count the overlap twice, so you must subtract it once. Formula: n(A∪B) = n(A) + n(B) − n(A∩B)
✅ Answer
n(A∪B) = n(A) + n(B) − n(A∩B)
n(A∪B) = 6 + 5 − 2 = 9
🎯 Practice Questions
📝 Try These
P1. n(A)=10, n(B)=8, n(A∪B)=15. Find n(A∩B).
n(A∩B) = n(A)+n(B)−n(A∪B) = 10+8−15 = 3
P2. In a class of 30 students, 18 study Maths, 15 study English, 7 study both. How many study at least one?
n(M∪E) = 18+15−7 = 26 students study at least one subject.
Q9 Prove A ∪ A = A (Idempotent Law) ▾
💡 What this is asking
This is the Idempotent Law. It means unioning a set with itself gives back the same set. No new elements can come from A that weren't already in A.
✅ Proof
  1. Suppose x ∈ A ∪ A. By definition of union, x ∈ A or x ∈ A. Either way, x ∈ A.
  2. Suppose x ∈ A. Then x ∈ A ∪ A (since x is in at least one of the A's).
∴ A ∪ A = A ✓
Also note: A ∩ A = A works the same way.
🎯 Practice Questions
📝 Try These
P1. Prove A ∩ A = A.
If x ∈ A∩A → x ∈ A and x ∈ A → x ∈ A. Conversely x ∈ A → x ∈ A and x ∈ A → x ∈ A∩A. ∴ A∩A = A ✓
P2. Prove A ∪ ∅ = A.
If x ∈ A∪∅ → x ∈ A or x ∈ ∅. Since ∅ has no elements, x must be in A. Conversely x ∈ A → x ∈ A∪∅. ∴ A∪∅ = A ✓
Q10 Find Complement A' given U={1..8}, A={1,3,5,7} ▾
💡 What this is asking
The complement A' (also written Aᶜ or Ā) = U − A. It's all elements in the universal set U that are NOT in A. Think of it as "everything else."
✅ Answer
U = {1,2,3,4,5,6,7,8}, A = {1,3,5,7}
A' = U − A = {2, 4, 6, 8}
(Everything in U that is NOT in A)
🎯 Practice Questions
📝 Try These
P1. U={a,b,c,d,e,f}, A={b,d,f}. Find A'.
A' = {a,c,e}
P2. Prove (A')' = A.
A' = U−A. (A')' = U−(U−A) = A. Since removing from U the things NOT in A leaves exactly A. ✓

Q11 Find probability of selecting a red ball from {3 red, 2 blue, 5 green} ▾
💡 What this is asking
Basic probability formula: P(event) = favourable outcomes divided by total outcomes. Count what you want, divide by everything possible.
✅ Answer
Total balls = 3+2+5 = 10. Red balls = 3.
P(red) = 3/10 = 0.3
P(red) = 3/10 = 30%

Total=12. P(blue)=6/12=1/2=0.5
P2. P(not red) from original bag of 10 (3 red)?
P(not red) = 1 − P(red) = 1 − 3/10 = 7/10
Q12 Find P(sum = 9) when rolling two dice ▾
💡 What this is asking
Two dice have 6×6=36 equally likely outcomes. List all pairs (die1, die2) that add up to 9, then divide by 36.
✅ Answer
Pairs summing to 9: (3,6),(4,5),(5,4),(6,3) = 4 pairs
P(sum=9) = 4/36 = 1/9
P(sum = 9) = 4/36 = 1/9 ≈ 0.111
🎯 Practice Questions
📝 Try These

Pairs: (1,6),(2,5),(3,4),(4,3),(5,2),(6,1) = 6 pairs. P = 6/36 = 1/6
P2. Find P(sum is even) with two dice.
Even sums: both even (3×3=9 ways) + both odd (3×3=9 ways) = 18 ways. P = 18/36 = 1/2
Q13 Probability of exactly 1 head when tossing 3 coins ▾
💡 What this is asking
3 coins gives 2³=8 equally likely outcomes. List all outcomes with exactly 1 head (H), count them, divide by 8. Each outcome is a sequence like HHT.
✅ Answer
All outcomes: HHH, HHT, HTH, HTT, THH, THT, TTH, TTT (8 total)
Exactly 1 head: HTT, THT, TTH = 3 outcomes
P(exactly 1 head) = 3/8

Q16 P(all balls blue) when drawing 2 from {4 blue, 3 red} without replacement ▾
💡 What this is asking
"Without replacement" means after you pick the first ball, you don't put it back. So the second pick changes. Use combinations: C(4,2)/C(7,2): ways to choose 2 blue from 4, divided by ways to choose any 2 from 7.
✅ Answer
P = C(4,2) / C(7,2) = 6/21 = 2/7
  1. C(4,2) = 4!/(2!2!) = 6 (ways to choose 2 blue)
  2. C(7,2) = 7!/(2!5!) = 21 (ways to choose any 2)
P(both blue) = 6/21 = 2/7 ≈ 0.286
🎯 Practice Questions
📝 Try These

Q17 Define Conditional Probability ▾
💡 What this is asking
Conditional probability P(A|B) means "the probability of A given that B has already happened." You restrict your sample space to only outcomes where B occurred.
✅ Definition and Formula
P(A|B) = P(A ∩ B) / P(B), where P(B) ≠ 0
Example: A card is drawn. Given it's a Heart, P(it's an Ace)?
P(Ace|Heart) = P(Ace AND Heart) / P(Heart) = (1/52)/(13/52) = 1/13
Conditional probability restricts the sample space to only events where B occurred.
🎯 Practice Questions
📝 Try These
P1. P(A)=0.4, P(B)=0.5, P(A∩B)=0.2. Find P(A|B).
P(A|B) = 0.2/0.5 = 0.4
P2. Roll a die. Given it shows even, what is P(it's a 4)?
Even numbers: {2,4,6}. Only one is 4. P(4|even) = 1/3
Q18 P(at least one tail) when tossing 2 coins ▾
💡 What this is asking
"At least one" is easiest using the complement rule: P(at least 1 tail) = 1 − P(no tails at all) = 1 − P(all heads).
✅ Answer
Outcomes: HH, HT, TH, TT (4 total)
P(no tail) = P(HH) = 1/4
P(at least 1 tail) = 1 − 1/4 = 3/4
Or count directly: HT, TH, TT = 3 outcomes → 3/4 ✓

Q20 Construct Probability Distribution for X = number of heads in 3 coin tosses ▾
💡 What this is asking
A probability distribution is a table listing all possible values of a random variable X and their probabilities. All probabilities must sum to 1.
✅ Answer
X can be 0, 1, 2, or 3 heads. Total outcomes = 8.
  1. TTT → 1 outcome → P = 1/8
  2. HTT,THT,TTH → 3 outcomes → P = 3/8
  3. HHT,HTH,THH → 3 outcomes → P = 3/8
  4. HHH → 1 outcome → P = 1/8
X: 0 1 2 3 P: 1/8 3/8 3/8 1/8 Sum = 1/8+3/8+3/8+1/8 = 8/8 = 1 ✓
🎯 Practice Questions

Q21 Find the Transpose of matrix A = [[1,2],[3,4],[5,6]] ▾
💡 What this is asking
The transpose Aᵀ flips a matrix: rows become columns and columns become rows. A 3×2 matrix becomes a 2×3 matrix.

A = [ [1,2], [3,4], [5,6] ] (3×2 matrix)
Aᵀ = [ [1,3,5], [2,4,6] ] (2×3 matrix) Row 1 of A becomes Column 1 of Aᵀ Row 2 of A becomes Column 2 of Aᵀ

Q27 Write the Identity Matrix I₂ and I₃ ▾

Q40 Explain WHY Binary Search needs sorted data ▾
💡 What this is asking
This is a conceptual question. You need to explain the fundamental logic of binary search and why it breaks on unsorted data.
✅ Answer
  1. Binary search works by comparing the target with the MIDDLE element, then eliminating HALF the remaining elements.
  2. This elimination only works if data is sorted. If arr[mid] < target, we KNOW all elements to the LEFT are also smaller. This assumption fails in unsorted data.
  3. Unsorted [3,9,1,7,5], search for 7: Mid=1(value=9), 9>7 → go left → [3] → not found! But 7 IS in the array at index 3.
Conclusion: Sorted order lets binary search make guaranteed decisions. Without it, eliminating halves is invalid and gives wrong answers. That is why we sort with Bubble Sort first.

Q50 Pigeonhole Principle: prove 2 people share a birth month in any group of 13 ▾
💡 What this is asking
The Pigeonhole Principle: if you put n+1 pigeons into n holes, at least one hole has 2+ pigeons. Here: 12 months = 12 "holes", 13 people = 13 "pigeons". One month must have 2 people!
✅ Answer
  1. There are 12 possible birth months (Jan to Dec). These are our "pigeonholes".
  2. With 13 people ("pigeons") and only 12 months ("holes"), by the pigeonhole principle at least one month must contain ⌈13/12⌉ = 2 people.
  3. If every month had at most 1 person → max 12 people. But we have 13, a contradiction!
General: n+1 items into n categories → at least one category has ≥2 items
∴ In ANY group of 13 people, at least 2 share a birth month. ✓ (QED)
