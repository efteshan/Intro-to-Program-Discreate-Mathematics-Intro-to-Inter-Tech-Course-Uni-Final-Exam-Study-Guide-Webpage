FM1 Must Master: Python Lists ▾
💡 What You Must Know
You must know how to create lists, add elements, iterate through lists, calculate total and average, find maximum and minimum, and filter elements based on conditions.
✅ Core Pattern
numbers = [10, 20, 30, 40]

total = sum(numbers)
avg = total / len(numbers)

for n in numbers:
    if n > avg:
        print(n)
Output
30 40
🔍 How it works
  • sum(numbers) adds all elements: 10+20+30+40 = 100
  • total / len(numbers) gives the average: 100/4 = 25.0
  • The for loop checks each number against the average
  • Only numbers greater than 25.0 (which are 30 and 40) get printed

FM2 Must Master: Tuples ▾
💡 What You Must Know
Tuples store fixed records that should not change. You must understand tuple unpacking, accessing tuple values, and using lambda functions to sort or find max/min in a list of tuples.
✅ Core Pattern: Tuple Unpacking
products = [
    ("Pen", 10),
    ("Book", 40),
    ("Bag", 500)
]

# Tuple unpacking in a loop
for name, price in products:
    print(name, price)
Output
Pen 10 Book 40 Bag 500
✅ Using Lambda to Find Most Expensive
most_expensive = max(products, key=lambda x: x[1])
print("Most expensive:", most_expensive)
Output
Most expensive: ('Bag', 500)
🔍 How lambda works here
  • lambda x: x[1] means "for each tuple x, look at index 1 (the price)"
  • max() compares all tuples by their price value
  • The tuple with the highest price (500) is returned

FM7 Sample Exam Problem: Student Marks Manager ▾
💡 Problem Statement
Write a Python program to manage student marks. Requirements: (1) Store student data using a dictionary. (2) Each record has ID, Name, and Marks. (3) Create a menu with: Add student, Search student, Display all students, Exit.
✅ Full Solution
students = {}

while True:
    print("\n--- Student Marks Manager ---")
    print("1. Add student")
    print("2. Search student")
    print("3. Display all students")
    print("4. Exit")
    choice = input("Enter choice: ")

    if choice == "1":
        sid = input("Enter ID: ")
        name = input("Enter Name: ")
        marks = int(input("Enter Marks: "))
        students[sid] = {"name": name, "marks": marks}
        print("Student added.")

    elif choice == "2":
        sid = input("Enter ID to search: ")
        if sid in students:
            info = students[sid]
            print("Name:", info["name"])
            print("Marks:", info["marks"])
        else:
            print("Student not found.")

    elif choice == "3":
        for sid, info in students.items():
            print(sid, info["name"], "Marks:", info["marks"])

    elif choice == "4":
        print("Program ended.")
        break
    else:
        print("Invalid choice.")
FM8 Common Errors to Avoid in the Exam ▾
⚠ Critical Mistakes That Cost Marks
Students often lose marks because of these common mistakes. Review each one carefully before the exam.
#Common ErrorHow to Avoid It
1Incorrect indentationPython uses indentation to define code blocks. Always use consistent 4-space indent inside loops, conditions, and functions.
2Missing append() when adding to listsYou must use list.append(item) to add items. Writing list + item does not modify the original list.
3Wrong dictionary key accessUse the exact key name: dict["name"] not dict[name] (without quotes it looks for a variable).
4Forgetting break in menu systemsWithout break in the exit option, your while True loop will run forever.
5Incorrect average calculationAverage = total / count. Use sum(list) / len(list). Make sure you do not divide by zero.

P1 Create a list of 5 numbers entered by the user and print the list ▾
💡 Concept
We create an empty list first, then use a for loop to ask the user 5 times. Each entered number is added to the list using .append().
✅ Solution
nums = []  # empty list
for i in range(5):
    n = int(input("Enter number: "))
    nums.append(n)
print("The list is:", nums)
Sample Output
The list is: [10, 20, 30, 40, 50]
🔍 How it works
  • nums = [] creates an empty list to hold values
  • range(5) repeats the loop exactly 5 times (i = 0,1,2,3,4)
  • int(input(...)) reads user input as a string, converts to integer
  • .append(n) adds each number to the end of the list

📌 Key Concept: What is a Tuple? ▾
A tuple is like a list but immutable, once created, you cannot change its elements. Use parentheses ( ). Perfect for fixed records like student ID, name, GPA.
Syntax & Access
# Create a tuple
student = ("S001", "Asha", 3.75)

# Access by index

print(student[1])   # "Asha"

# Slicing

# This CAUSES AN ERROR (tuples are immutable):
# student[1] = "Rahim"  ← TypeError!
List vs Tuple: Use a list when data needs to change (e.g. marks being added). Use a tuple for fixed records that should not change (e.g. student ID + name).

P7 Tuple slicing: create a tuple of 5 cities, print first city and last two ▾
💡 Concept
Slicing extracts a portion of a tuple. tuple[-2:] means "from 2nd-last to end." Negative indexes count from the right: -1 = last, -2 = second-to-last.
✅ Solution
cities = ("Dhaka", "Chittagong", "Rajshahi", "Khulna", "Sylhet")
print("First city:", cities[0])
print("Last two cities:", cities[-2:])
Output
First city: Dhaka Last two cities: ('Khulna', 'Sylhet')
🔍 How slicing works
  • Index 0 → first item → "Dhaka"
  • Index -1 → last item → "Sylhet"
  • Slice -2: → last 2 items → ("Khulna", "Sylhet")
  • Slice 1:3 → index 1 up to (not including) 3 → ("Chittagong", "Rajshahi")

Part C: Dictionaries
Key-value mapping · Problems 9-13
📌 Key Concept: What is a Dictionary? ▾
A dictionary stores data as key: value pairs inside curly braces { }. Instead of using an index number like a list, you look up values using a meaningful key (like a name or ID). Perfect for fast lookup.
Syntax & Operations
# Create
student = {"name": "Asha", "marks": 85}

# Access by key
print(student["name"])      # "Asha"
print(student.get("age", "N/A"))  # safe access

# Add / update
student["grade"] = "A"     # add new key
student["marks"] = 90     # update existing

# Loop through
for key, value in student.items():
    print(key, ":", value)

P11 Count word frequency in a sentence using a dictionary ▾
💡 Concept
Use the word as the key and its count as the value. For each word: if it's already in the dictionary, add 1 to its count. If it's new, set its count to 1.
✅ Solution
sentence = input("Enter sentence: ").lower()
words = sentence.split()
freq = {}

for w in words:
    if w in freq:
        freq[w] += 1
    else:
        freq[w] = 1

print(freq)
Sample (input: "I love python python is fun")
{'i': 1, 'love': 1, 'python': 2, 'is': 1, 'fun': 1}

📌 Key Concept: Functions in Python ▾
A function is a reusable block of code with a name. You define it once with def, then call it many times. Functions can accept data (parameters) and send back results (return).
Function Anatomy
def function_name(parameter1, parameter2):
    # code goes here
    return result

# Calling the function
answer = function_name(value1, value2)
print(answer)
Why use functions? Avoid repeating the same code. Break a big problem into small, named, testable pieces. Makes your program easier to read and fix.

CA3 Understanding Lambda Functions ▾
💡 Concept
A lambda is a short, one-line function without a name. It is often used inside max(), min(), or sorted() to tell Python which value to compare by. The syntax is lambda parameter: expression.
✅ Exam-style Example
products = [("Pen", 10), ("Bag", 500)]
result = max(products, key=lambda x: x[1])
print(result)
Output
('Bag', 500)
🔍 How to read lambda
  • x represents each tuple in the list, one at a time
  • x[1] means "use the second value of the tuple" (the price)
  • max() compares all tuples by their price and returns the biggest
  • So ("Pen", 10) vs ("Bag", 500) compares 10 vs 500, and returns Bag

S2 Menu-based Student Manager: full interactive program ▾
💡 Concept
A menu-driven program uses a while True loop to keep running, showing a menu each time. The user picks a number, and an if/elif/else chain runs the right action. break exits the loop.
✅ Full Program
students = {}

while True:
    print("\n--- Student Manager ---")
    print("1. Add student")
    print("2. Add marks")
    print("3. Search student")
    print("4. Display all")
    print("5. Exit")
    choice = input("Enter choice: ").strip()

    if choice == "1":
        sid  = input("ID: ").strip()
        name = input("Name: ").strip()
        students[sid] = {"name": name, "marks": []}
        print("Added.")

    elif choice == "2":
        sid = input("Enter ID: ").strip()
        if sid in students:
            m = int(input("Enter mark: "))
            students[sid]["marks"].append(m)
            print("Mark added.")
        else:
            print("Student not found.")

    elif choice == "3":
        sid = input("Enter ID: ").strip()
        if sid in students:
            print(sid, students[sid])
        else:
            print("Not found.")

    elif choice == "4":
        for sid, info in students.items():
            marks = info["marks"]
            avg = sum(marks)/len(marks) if marks else 0
            print(sid, info["name"], "Marks:", marks, "Avg:", avg)

    elif choice == "5":
        print("Program ended.")
        break
    else:
        print("Invalid choice.")
🔍 How each part works
  • Loop runs forever until break is hit (option 5)
  • Creates a new entry in the students dict with empty marks list
  • Finds student by ID, appends mark to their list
  • Looks up student by ID, instant access with dict
  • Loops all students, computes avg on the fly
  • avg = ... if marks else 0 prevents division by zero if no marks yet
